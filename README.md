# Reveal

Install Reveal via Boxen (http://www.revealapp.com)

## Usage

```puppet
include reveal
```

## Required Puppet Modules

* `boxen`

[![Build Status](https://travis-ci.org/crafterm/puppet-reveal.png?branch=master)](https://travis-ci.org/crafterm/puppet-reveal)

Write code.
